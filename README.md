# Operating System Visual Learning Platform

A premium educational React + Vite application for university students studying operating systems and CPU architecture.

## Features
- React 19 + Vite frontend
- React Router navigation
- Zustand state management
- Tailwind CSS styling with glassmorphism and dark theme
- Framer Motion animations
- React Three Fiber 3D visualizations
- Interactive simulators for CPU, process, thread, and instruction concepts
- Responsive layout for desktop, tablet, and mobile screens
- Search bar and progress tracking foundation

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
- The app includes interactive educational pages for CPU architecture, process scheduling, thread synchronization, and instruction execution.
- The project structure is modular and best suited for further extension with additional visual simulations and content.
